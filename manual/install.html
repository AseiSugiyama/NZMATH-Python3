Basic installation †

To install NZMATH, you must have a Python 2.3 installed in your computer or a higher version. If you don't have a copy of Python, please install it first. Python is available from www.python.org .

The next step is to extract NZMATH-0.5.1.tar.gz. You can easily unzip this, depending on your OS. For systems with recent GNU tar, you can do it with a single command:

% tar xf NZMATH-0.5.1.tar.gz

where, % is the command line prompt. Or with standard tar, you can do it as:

% gzip -cd NZMATH-0.5.1.tar.gz | tar xf -

Then, you have a child directory named NZMATH-0.5.1.

The last step is to install NZMATH to the standard python path. Usually, this translates to writing files somewhere under /usr/lib or /usr/local/lib, and thus you have to have appropriate write permission. Typically, do as the following:

% cd NZMATH-0.5.1
% su
# python setup.py install

Package installations †

These packages may be updated after our releases.

↑

KNOPPIX/Math †

KNOPPIX/Math 2006 is shipped with NZMATH-0.4.1 installed.

↑

Gentoo Linux †

There is a private package for Gentoo Linux by Matsui Tetsushi. Put nzmath-0.5.0.ebuild into your overlay portage tree of category sci-mathematics. Then emerge it as usual if you are of ~x86 user. Other arch user have to cast a spell on your /etc/portage or the ebuild.

↑

Mac OS X (Tiger) †

A NZMATH 0.5.0 application binary for Mac OS X NZMATH-20060225.tar.gz is contributed by Kishi Toshiharu. You can use it by following instruction:

  1. download it.
  2. expand it.
  3. move resulting NZMATH.app to Application Folder.
  4. double click the NZMATH.app.

Note that it works on Tiger (10.4) / PPC only.
