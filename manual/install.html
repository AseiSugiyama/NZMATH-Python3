  • Basic installation

Basic installation †

To install NZMATH, you must have a Python 2.3 installed in your computer or a higher version. If you don't have a copy of Python, please install it first. Python is available from http://www.python.org/ .

The next step is to extract NZMATH-0.4.0.tar.gz. You can easily unzip this, depending on your OS. For systems with recent GNU tar, you can do it with a single command:

% tar xf NZMATH-0.4.0.tar.gz

where, % is the command line prompt. Or with standard tar, you can do it as:

% gzip -cd NZMATH-0.4.0.tar.gz | tar xf -

Then, you have a child directory named NZMATH-0.4.0.

The last step is to install NZMATH to the standard python path. Usually, this translates to writing files somewhere under /usr/lib or /usr/local/lib, and thus you have to have appropriate write permission. Typically, do as the following:

% cd NZMATH-0.4.0
% su
# python setup.py install

Package installations †

Gentoo Linux †

There is a private package for Gentoo Linux. Put nzmath-0.4.0.ebuild into your overlay portage tree of category sci-mathematics. Then emerge it as usual if you are of ~x86 user. Other arch user have to cast a spell on your /etc/portage or the ebuild.
