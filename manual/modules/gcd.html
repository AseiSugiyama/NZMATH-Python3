gcd module †

The module provides functions to compute greatest common divisors of integers.

Functions †

gcd(a,b) †

Returns the greatest common divisor of 2 integers a and b.

binarygcd(a, b) †

Returns the greatest common divisor of 2 integers a and b by binary gcd algorithm.

extgcd(x,y) †

Returns the greatest common divisor d of two integers x and y and u, v such that d = x * u + y * v. The returned value is a tuple (u, v, d).

lcm(a,b) †

Returns the least common multiple of 2 given integers a and b. If both entries are zero, then it raises an exception.

gcd_of_list(integers) †

Returns a list [d, [c1, ..., cn]] for the given list integers [x1, ..., xn] such that
d = c1 * x1 + ... + cn * xn.
