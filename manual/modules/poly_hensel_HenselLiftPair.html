HenselLiftPair †

A class represents integer polynomial pair which will be lifted by Hensel's lemma, so-called the quadratic method.

Target and all its factors are supposed to be monic.

Constructor †

HenselLiftPair(f, a1, a2, u1, u2, p, q) †

The parameters satisfy that:

a1 and a2 are monic,
f == a1*a2 (mod q) and
a1*u1 + a2*u2 == 1 (mod p),

with positive integers p dividing q.

If p==q, q can be omit from the argument.

from_factors(f, a1, a2, p) †

A class method to create and return an instance of HenselLiftPair.

The parameters satisfy that:

f == a1*a2 (mod p)

with a prime number p.

Methods †

lift_factors() †

Update factors by lifted integer coefficient polynomials Ai's:

f == A1*A2 (mod p*q) and
Ai == ai (mod q) (i = 1, 2).

Moreover, q is updated by p*q

PRECONDITIONS (automatically satisfied):

f == a1*a1 (mod q)
a1*u1 + a2*u2 == 1 (mod p)
p | q

lift_ladder() †

Update u1 and u2 with U1 and U2:

a1*U1 + a2*U2 == 1 (mod p**2)
Ui == ui (mod p) (i = 1, 2)

then, update p with p**2.

PRECONDITIONS (automatically satisfied):

a1*u1 + a2*u2 == 1 (mod p)

lift() †

The lift. You should call this method.
