Class EC †

↑

Initialize †

How to asssign curves in Weierstrass form given by the equation:
E:

y^2 + a1xy + a3y = x^3 + a2x^2 + a4x + a6

>>> E=EC([a1,a2,a3,a4,a6],p)

or for

y^2= x^3 + a4x + a6

>>> E=EC([a,b],p)

If the parameter p is not given (p=None) then E is defined over Q.

↑

Notation †

Note the points on E are represented as two-component vectors [x,y], except the point at infinity which is represented by the one-component vector [0] .
i.e., infinity point ([0, 1, 0]), as follows:

>>> infpoint = [0]
↑

Methods †

The following are the implemented methods in these module.

E.ch
returns the characteristic of the field Fp. For this case, it gives ch=p .
E.a1, E.a2, E.a3, E.a4, E.a6
returns the coefficients a1, a2, a3, a4, a6, resp.
E.b2, E.b4, E.b6, E.b8
returns the coefficents b2, b4, b6, b8 .
E.c4, E.c6
returns the coefficients c4 and c6 .
E.disc
returns the discriminant of the curve E .
E.j
returns the j-invariant of the curve E .
E.simple()
changes the data of E given in the short Weierstrass form by changing the coordinates if E.ch>3 .
E.changeCurve(V)
changes the data of E by changing the coordinates using the vector V=[u,r,s,t] with u!= 0. i.e., if x' and y' are the new coordinates, then y^2 + a1xy + a3y = x^3 +a2x^2 + a4x + a6
E.changePoint(P,V)
changes the coordinates of the point using the vector V=[u,r,s,t], i.e., if x' and y' are the new coordinates, then y^2 + a1xy + a3y = x^3 +a2x^2 + a4x + a6 by an admissible change in variables.
E.point()
returns a random point on E.
E.coordinateY(x)
returns a 1 or 2-component vector containing the y-coordinates of the points of E having x as x-coordinate.
E.whetherOn(P)
returns True if the point P is on E, False otherwise.
E.add(P,Q)
returns the sum of points P and Q on E.
E.sub(P,Q)
returns the difference of points P and Q on E.
E.mul(k,P)
returns k times P.
E.divPoly(m)
computes the mth divisor polynomial. If m is odd, this method returns the usual division polynomial. If m is even, it returns the usual division polynomial divided by 2y + a1x + a3 . If the parameter m is not specified (m=None), then this returns the list of division polynomials for Schoof's algorithm.

The following methods have been implemented only for elliptic curves over prime fields Fp except for 2 and 3.

E.Schoof()
computes t such that #E(Fp)=p+1-t, where #E(Fp) stands for the number of points of E over Fp. This uses the Schoof's method.
E.Shanks_Mestre()
computes t such that #E(Fp)=p+1-t, where #E(Fp) stands for the number of points of E over Fp. This uses the Shanks-Mestre method.
E.naive()
computes t such that #E(Fp)=p+1-t, where #E(Fp) stands for the number of points of E over Fp. This uses the Legendre-symbol method.
E.order(flag=0)
computes #E(Fp). If p<1000, this uses the Legendre method. If 10^4<p<10^30, this uses the Shanks-Mestre method. Otherwise, it uses Schoof's method. If flag=r, this computes #E(Fp^r) .
E.Miller(P,m,Q)
returns the f_D_P such that (f_D_P)=D_P with P,Q in E[m] by using Miller's algorithm.
E.WeilPairing(m,P,Q)
returns e_m(P,Q) by using E.Miller(P,m,Q).
E.BSGS(n,P,Q)
returns m such that Q=mP. P,Q are in the subgroup of order n.
E.structure()
returns the group structure (n1, n2) of the E(Fp) by using Miller's algorithm.
