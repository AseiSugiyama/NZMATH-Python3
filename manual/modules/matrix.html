matrix †

A=Matrix(m,n,[a11,a12,...a1n,a21,...,am1,...amn])

For example:

>>> A = Matrix(2,2, [1,2,5,10])

yields

>>> A
1  2 
5  10

It is also possible to use the following syntax:

B=Matrix(n,k,[b11,..,b1k]+....+[bn1,...,bnk])

For example:

>>> B = Matrix(2,3, [0,1,3]+[0,2,2])

giving:

>>> B
0 1 3
0 2 2

It is possible to do operations on the matrices. Addition (+), Subtraction (-), Multiplication (*), as well as LU Decomposition:

>>> M.LUDecomposition()

In the example, we should get:

( 1 0
3/1 1 ,
1   2
0/1 4/1 )
