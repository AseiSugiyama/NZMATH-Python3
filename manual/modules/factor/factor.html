factor module †

factor module provides functions of factorization .

functions †

rhomethod(n) †

Factor an integer n by Pollard's rho method. The method is preferable for numbers having small factors. But the method is a probabilistic method, possibly fails in factorizations.

  • subrhomethod(n)
    This function is to find a non-trivial factor of n using algorithm of Crandall&Pomerance
  • sort_factor(list)
    Sort factors for rho methods. (will be deleted in the next release 0.4.1)
  • FactoringIntegerForRhoMethod
    The class keeps track of the rho method factorization. (will be added in 0.4.1)

pmom(n) †

Factor an integer n by p-1 method. The method is preferable for numbers having small factors.

  • subpmom(n)
    This function tries to find a non-trivial factor of N using Algorithm 8.8.2 (p-1 first stage) of Cohen's book. In case of N = pow(2,i), this program willnot terminate.

mod_sqrt(a,m) †

Return a square root of a mod m.

ord(p,n) †

Return the power of the prime p which divides the non-zero integer n.
(will be removed in the next release (0.4.1). Please use arith1.vp instead.)

primeDivisors(n) †

primeDivisors(n) returns the list of primes that divides n.

PrimePowerTest(n) †

This function using Algo. 1.7.5 in Cohen's book judges whether n is of the form p**k with p or not. If it is True, then returned value is (p,k). Otherwise, returned value is (n,0).

squarePart(n) †

squarePart(n) returns the largest integer whose square divides n.

AllDivisors(n) †

Return all divisors of n.
