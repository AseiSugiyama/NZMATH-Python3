 有理関数は二つの多項式の,分数として書けるもの.

 このモジュールが役に立つと期待しないこと.
 ただ多項式の除算のための無難なコンテナを提供するもの.
\C

 \subsection{RationalFunction -- 有理関数クラス}\linkedone{poly.ratfunc}{RationalFunction}
 \initialize
  \func{RationalFunction}{\hiki{numerator}{polynomial},\ \hikiopt{denominator}{polynomial}{1}}{\out{RationalFunction}}\\
  \spacing
  % document of basic document
  \quad 与えられた\param{numerator}と\param{denominator}を持つ有理関数を作る.
  % added document
  %
  % \spacing
  % input, output document
  もし\param{numerator}が{\tt RationalFunction}のインスタンスで,\param{denominator}が与えられなければコピーを作る.
  もし\param{numerator}が多項式なら,numeratorが与えられた有理関数を作る.
  さらに,もし\param{denominator}がすでに与えられていたら,分母はその値で設定され,さもなくば分母は1.
  \begin{at}
    \item[numerator]\linkedtwo{poly.ratfunc}{RationalFunction}{numerator}:\\
      多項式.
    \item[denominator]\linkedtwo{poly.ratfunc}{RationalFunction}{denominator}:\\
      多項式.
  \end{at}
  \begin{op}
    \verb+A==B+ & {\tt A}と{\tt B}が等しいかどう返す.\\
    \verb+str(A)+ & 読みやすい文字列を返す.\\
    \verb+repr(A)+ & {\tt A}の構造表現文字列を返す.\\
  \end{op}

   \spacing
   % document of basic document
   \quad 有理関数が所属する有理関数体を返す.
