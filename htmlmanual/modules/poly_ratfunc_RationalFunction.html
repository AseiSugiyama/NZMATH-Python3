RationalFunction(numerator, [denominator]) †

Make a rational function with the given numerator and denominator. If the numerator is a RationalFunction instance and denominator is not given, then make a copy. If the numerator is a kind of polynomial, then make a rational function whose numerator is the given polynomial. Additionally, if denominator is also given, the denominator is set to its valus, otherwise the denominator is 1.
