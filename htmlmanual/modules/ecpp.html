nzmath

ecpp †

The module consists of various functions for ECPP (Elliptic Curve Primality Proving).
(new in 0.92.0)

It is probable that the module to be refactored in the future so that each function be placed in other modules.

Requirement †

The ecpp module needs mpmath.

Functions †

ecpp(n [,era]) †

If n is prime, retrun True. Else, return False. n should be a big integer.

The optional argument era is a list of primes (stands for ERAtosthenes).

This is the main function in the module.

hilbert(D) †

Return (the coefficients of) Hilbert class polynomial for the imaginary quadratic field of fundamental discriminant D.

dedekind(tau, floatpre) †

Return Dedekind's eta of a complex number tau in the upper half-plane. Additional argument floatpre specifies the precision of calculation in "decimal digits".

cmm(p) †

Return curve parameters for CM curves (a, b of Weierstrass' short form). A prime p has to be odd.

A cousin function cmm_order(p) returns the order of curve as well.

cornacchiamodify(d, p) †

Return the solution (u, v) of u**2 + d * v**2 = 4p. The parameters satisfy that p be a prime and d be an integer such that 0 < d < 4p with -d = 0, 1 (mod 4). If there is no solution, raise ValueError.
