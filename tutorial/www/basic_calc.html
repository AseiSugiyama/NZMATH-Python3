基本的な数値計算

数値の表現方法

■整数
Python組込みのInteger型を使います。
> number = 12345
> print(number)
12345

■有理数（分数）
NZMATHのRationalクラスのインスタンスとして表します。
> from nzmath.rational import *
> a = Rational(2, 3)　　　　// a に 2/3 をセット
> b = Rational(1, 3)　　　　// b に 1/3 をセット
> a + b
Rational(1, 1)
> a * b
Rational(2, 9)
→ 詳細はこちら

■実数(近似値)
Python組込みのFloat型を使います。
> number = 123.45
> print(number)
123.45

■複素数
NZMATHのComplexクラスのインスタンスとして表します。
> from nzmath.imaginary import *
> a = Complex(2, 3)　　　　// a に 2+3i をセット
> b = Complex(0, 1)　　　　// b に i をセット
> a + b
Complex(2, 4)
> a * b
Complex(-3, 2)
→ 詳細はこちら

演算

■四則演算
NZMATHに含まれるクラスは、加減乗除がそれぞれ +, -, *, / で行えるように演算子が定義されています。
> 2 + 3　　// 5
> 2 - 3　　// -1
> 2 * 3　　// 6
> 2 / 3　　// 0 (Integerの除算では、小数点以下は切り捨てられます。)
> 2 % 3　　// 2 (2÷3の剰余)

■冪乗
** 演算子、または、Python組込みのpow関数で計算できます。
> 2 ** 10　　// 2 の 10 乗
1024
> pow(2, 10)
1024

mod m での冪乗を計算したい場合には、pow関数の第3引数にmを指定します。
> pow(2, 10, 100)
24
