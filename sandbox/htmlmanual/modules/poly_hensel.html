hensel †

A module for Hensel lift.
(new in 0.92.0)

Generally, users only need the function list_upto.

Note †

A basic implementation of Hensel lift has been there in poly.factor.

functions †

lift_upto(target, factors, p, bound) †

Hensel lift factors mod p of target upto bound and return factors mod q and the q itself.

PRECONDITIONS:

target is monic.
target == product(factors) mod p

POSTCONDITIONS:

there exist k s.t. q == p**k >= bound and
target == product(result) mod q
