EC(Weierstrass list form, basefield) †

A factory function to create an instance of ECoverQ or ECoverGF. and their are subclass of ECGeneric, see also as use.
If basefield is a object inheriting finitefield.FiniteField or a prime number, create elliptic curve over finite field(ECoverGF). Otherwise (if basefield is not given or object of rational.RationalField or 0), create elliptic curve over rational field(ECoverQ) .

How to asssign curves in Weierstrass list form given by the equation:

E: y^2 + a1 xy + a3 y = x^3 + a2 x^2 + a4 x + a6

>>> E=EC([a1,a2,a3,a4,a6],basefield)

or for

E: y^2= x^3 + A x + B

>>> E=EC([A,B],basefield)
